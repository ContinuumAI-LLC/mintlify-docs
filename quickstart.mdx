## Requisitos previos

Antes de comenzar, asegúrate de tener:

- Node.js 18 o superior
- Una cuenta activa en el proyecto
- Tu clave de API lista

## Instalación

<Steps>
  <Step title="Instala el paquete">
    Agrega la dependencia a tu proyecto con tu gestor de paquetes favorito.

    ```bash
    npm install mi-paquete
    ```
  </Step>
  <Step title="Configura tus credenciales">
    Crea un archivo `.env` en la raíz del proyecto y agrega tu clave de API.

    ```bash
    API_KEY=tu_clave_aqui
    ```
  </Step>
  <Step title="Ejecuta el proyecto">
    Levanta el servidor y verifica que todo esté funcionando correctamente.

    ```bash
    npm run dev
    ```
  </Step>
</Steps>
